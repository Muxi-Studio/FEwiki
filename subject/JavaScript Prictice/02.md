## 02 实现字符串Padding

### 要求

实现一个JavaScript`String`类型的方法`padStart`。`padStart`接收两个参数, `targetLength`和`padString`。其中`padString`是可选的。这个函数的作用是在字符串的开始处添加`targetLength`个`padString`。`padString`默认为`" "`（(U+0020)。要求时间复杂度为线性。


### 示例输入

```
"Hello world".padStart(2)
```

### 示例输出

```
"  Hello world"
```
